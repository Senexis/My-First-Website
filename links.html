Sublime Text 3

Sublime Text is een tekstbewerkingsprogramma om handig HTML-pagina’s te kunnen schrijven. Sublime Text werkt ongeveer hetzelfde als Small Basic in het opzicht dat commando’s zoals “html” automatisch aangevuld worden. Ook worden argumenten automatisch weergeven zodat je tussen de verschillende argumenten kan kiezen, zodat je altijd alle commando’s goed heb getypt. Ook wordt de gehele code aan de rechterkant van het venster weergeven, waardoor je gemakkelijk kan zien wat waar staat, en kan je er heel gemakkelijk doorheen scrollen. Er zijn ook meerdere kleuren, waardoor je gemakkelijk de ene code van de andere kan onderscheiden.


Behalve voor het handig schrijven van HTML-pagina’s, kun je er ook – op dezelfde manier – gemakkelijk CSS-pagina’s mee schrijven. Er is ook een handig overzicht om tussen verschillende bestanden te kunnen switchen. Dit tab-systeem werkt hetzelfde als de populaire internetbrowser Google Chrome.


Sublime Text is een trailware. Dit betekend dat dit géén gratis software is, maar er wel een demo van is, zodat je het uit kunt proberen en kunt zien of je het programma leuk vindt of niet.

SkyDrive


SkyDrive is een handig programma om bestanden op je computer te synchroniseren met de cloud. Dit zal ervoor zorgen dat al je bestanden veilig overal toegankelijk zijn, waardoor je nooit meer met externe apparaten moet werken. Het enige wat je moet doen is het programma installeren en inloggen met je Outlook-account. Het programma is gemaakt door Microsoft, dus het zou kunnen zijn dat het zelfs al geïntegreerd zit binnenin Windows zelf!


Zie hoe je het moet gebruiken, vele tips en alles wat je erover moet weten op de website die hieronder vermeld staat. Bekijk ook meteen het inleidende filmpje over SkyDrive!

Oracle VM VirtualBox

Ocarcle VM VirtualBox is een handige software om een virtuele computer aan te maken. Het is enorm simpel om zo’n virtuele computer aan te maken. Ook kan je te allen tijde de instellingen aanpassen aan wat je nodig heb. Het programma heb je nodig als je bijvoorbeeld een nieuw besturingssysteem wilt uittesten maar niet op je computer wilt installeren en zo mogelijk gegevens kan verliezen.


Het is ook niet gelimiteerd, wat betekend dat je zoveel virtuele computers aan kan maken als jouw computer dat aankan. Het programma is gratis te gebruiken. Het enige wat je nodig heb is het programma en een installatie-cd van het besturingssysteem dat je wilt installeren. Dit laatste is nodig om bepaalde redenen. Het is dus geen gratis toegang naar de nieuwste besturingssystemen, puur en alleen een virtuele computer.


Het is ook mogelijk om de virtuele computer met je eigen besturingssysteem te integreren, hoewel er wel beperkingen zijn in welke besturingssystemen je kunt integreren. Als je er een hebt die je kunt integreren met je eigen besturingssysteem, zal je vlekkeloos van de ene naar de andere kunnen switchen. In feiten, hoef je in sommige gevallen niet eens te switchen en kan je gewoon alles direct op de virtuele computer doen vanuit je eigen besturingssysteem.


Ga naar de website van Oracle VM Virtualbox.
